# 🎓 Alumni Portal – Full-Stack Production Application

A modern, feature-rich Alumni Portal built with Next.js 14, TypeScript, Tailwind CSS, Framer Motion, Prisma, and PostgreSQL.

npm run db:seed
npm run dev
